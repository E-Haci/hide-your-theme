WP detectors will not be able to detect which theme your website are using. And your theme will not be renamed. The plugin will just hide its name.

== Description ==

“Hide your theme name” plugin is the WordPress plugin which helps you to hide your active theme name from all WP Detector websites. And you can do this without any knowledge about Apache, PHP, WordPress Codex, MySQL etc… Everything is simple.
Install -> Open Settings page -> One single click -> See the result and check it in WP detector websites.



You can also strip html comments from source code and to compress output html 


= Warning =
<b>If you meet any problem after installing, please don't panic, just uninstall the plugin and it will be ok. No any data lost will happen. 
But if you can't even access wp-admin, then just remove the plugin via ftp, then open wp-admin->settings->permalinks, press to "Save changes" button. Then write to our support email.

== Installation ==

1. Upload 'guaven_theme_name_hider' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Dashboard/Settings/Hide your theme name for configuring ”Hide your theme name”. 
4. Give any pseudo-name to your theme(it will not rename your theme)
